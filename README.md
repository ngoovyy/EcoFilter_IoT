# 🌊 EcoFilter - IoT Dashboard

### 📌 Giới thiệu dự án
Hệ thống giám sát bẫy vi nhựa thời gian thực dựa trên nền tảng IoT, thuộc dự án nghiên cứu Khoa học Kỹ thuật (KHKT) nhằm đánh giá hiệu suất giữ lại vi nhựa của vật liệu màng sinh học Keratin.

### 🚀 Tính năng cốt lõi
* **Giám sát lưu lượng:** Đo đạc chính xác tổng lượng nước đã đi qua bẫy lọc thông qua cảm biến dòng chảy ESP32.
* **Thuật toán ước tính:** Tự động tính toán khối lượng vi nhựa (mg) được giữ lại dựa trên hiệu suất màng lọc.
* **Cảnh báo bão hòa:** Hiển thị trực quan mức độ bão hòa của màng Keratin để người dùng kịp thời thay thế, đảm bảo an toàn nguồn nước.

### 🛠️ Công nghệ sử dụng
* **Front-end:** HTML5, CSS3 (Dark Green Theme), JavaScript (ES6)
* **IoT & Cloud:** ESP32 NodeMCU, Google Firebase Realtime Database
* **Version Control:** Git & GitHub Pages
